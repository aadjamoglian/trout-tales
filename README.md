# trout-tales
A fish tracker app that allows users to log their catches and a a shorty story of their catch.

At the very least the application should be able to display all of a users catches to them in a table. This will include information like date, species, weight, length, and location.

Location is where this could get intersting because we could use lat and long and then potentially map these coordiantes and display a map to users of all their catches!
